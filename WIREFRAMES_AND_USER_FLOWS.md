## User Experience Considerations

### Loading States

1. **CV Upload**: Show progress bar during upload
2. **CV Analysis**: Show spinner with estimated time
3. **Recommendations**: Show skeleton loaders while fetching

### Error Handling

1. **Upload Errors**: 
   - File too large â†’ Show error message
   - Unsupported format â†’ Show format requirements
   - Network error â†’ Retry button

2. **Analysis Errors**:
   - Extraction failed â†’ Show error, allow re-upload
   - Low confidence â†’ Show warning, allow manual edit

3. **Recommendation Errors**:
   - No recommendations â†’ Show message, suggest updating profile
   - API error â†’ Show error message, retry button

### Success States

1. **CV Uploaded**: Show success message, redirect to analysis
2. **Analysis Complete**: Show extracted data, allow confirmation
3. **Recommendations Generated**: Highlight new matches
4. **Job Saved**: Show confirmation, update UI
